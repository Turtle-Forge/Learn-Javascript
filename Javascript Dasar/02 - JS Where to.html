JavaScript Functions & Events

A JavaScript function is a block of JavaScript code, that can be executed when "called" for.
For example, a function can be called when an event occurs, like when the user clicks a button.

JavaScript in <head> or <body>

Anda dapat menempatkan sejumlah skrip dalam dokumen HTML.
Skrip dapat ditempatkan di bagian <body>, atau di bagian <head> pada halaman HTML, atau di keduanya.

External JavaScript

Skrip juga dapat ditempatkan dalam file eksternal.
Skrip eksternal sangat praktis ketika kode yang sama digunakan di banyak halaman web yang berbeda.
File JavaScript memiliki ekstensi file .js.
To use an external script, put the name of the script file in the src (source) attribute of a <script> tag:

Note: Skrip eksternal tidak dapat berisi tag <script>.

Contoh

Helo, belajar JavaScript menyenangkan

Keuntungan JavaScript Eksternal

Menempatkan skrip dalam file eksternal memiliki beberapa keuntungan:

  • Memisahkan HTML dan kode
  • Membuat HTML dan JavaScript lebih mudah dibaca dan dipelihara
  • File JavaScript yang di-cache dapat mempercepat pemuatan halaman

Untuk menambahkan beberapa file skrip ke satu halaman - gunakan beberapa tag skrip:

contoh
<script src="myScript1.js"></script>
<script src="myScript2.js"></script>
