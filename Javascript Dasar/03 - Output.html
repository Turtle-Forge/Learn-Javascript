Kemungkinan Tampilan JavaScript

JavaScript dapat "menampilkan" data dengan berbagai cara:

  • Menulis ke dalam elemen HTML, menggunakan innerHTML.
  • Menulis ke dalam output HTML menggunakan document.write().
  • Menulis ke dalam kotak peringatan, menggunakan window.alert().
  • Menulis ke dalam konsol peramban, menggunakan console.log().

Menggunakan innerHTML

Untuk mengakses elemen HTML, JavaScript dapat menggunakan metode document.getElementById(id).
Atribut id mendefinisikan elemen HTML. Properti innerHTML mendefinisikan konten HTML:

Contoh

10 + 20 = ...


Tips: Mengubah properti innerHTML dari sebuah elemen HTML adalah cara yang umum digunakan untuk menampilkan data dalam HTML.

JavaScript tidak memiliki objek print atau metode print.
Anda tidak dapat mengakses perangkat output dari JavaScript.
Satu-satunya pengecualian adalah Anda dapat memanggil metode window.print() di browser untuk mencetak konten jendela saat ini.
